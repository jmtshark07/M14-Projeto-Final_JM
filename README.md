# M14-Projeto-Final_JM
Projeto Final módulo 14

-Explicação de todas as fases do projeto
  -Primeira Fase do nosso projeto foi escolher o tema stock.
  -Segunda Fase Criamos a Base de dados no DbReaver e criamos as tabelas.
  -Terceira Fase Fizemos as ligações das Tabelas.
  -Quarta Fase Inserimos os dados nas Tabelas.
  -Quinta Fase fizemos o Sistema de Segurança.
  -Sexta Fase Fizemos as Consultas. 
-Justificação das decisões do design 
  A base de dados Foi criada a partir da interface e as tabelas o mesmo Pois tivemos dificuldades com o código e não estava a rodar como deve ser 
  O inserir dados , Sistema de Segurança, e as consultas foram em código Porque facilitou nos no nosso projeto 
-Instruções para replicar a bases de dados
